# 📊 Обучение статистике (Часть 1)

Репозиторий для фиксации прогресса и практики в курсе по основам статистики. Здесь я веду конспекты и отрабатываю навыки анализа данных с использованием Python.

## 🎓 О курсе
Я прохожу обучение на платформе **Stepik** в рамках курса от **Института биоинформатики**:
* **Название:** [Основы статистики. Часть 1](https://stepik.org/course/76)

---

## 📂 Структура репозитория

* `Statistics.ipynb` — конспекты по теории, примеры кода для регрессионного анализа и визуализации данных.
* `main.py` — дополнительные скрипты для автоматизации статистических расчетов.
* `Задачи.ipynb` — рабочая тетрадь, где я решаю практические задачи и упражнения по пройденным темам.

---

* **Язык:** Python 3
* **Основные библиотеки:** `pandas`, `numpy`, `scipy.stats`, `statsmodels`, `matplotlib`, `seaborn`.
* **Среда разработки:** Google Colab / PyCharm.
